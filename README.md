# Global Voices Technical Assignment

## Description
**Global Voices Technical Assignment** is a user authentication system designed to manage login, registration, and logout functionalities. This project ensures the secure storage of user credentials and implements token-based authentication, focusing on both security and efficiency.

## Built With
- **Front End**: React, Material-UI (MUI), Redux
- **Back End**: Node.js with Apollo, GraphQL, Mongoose
- **Database**: MongoDB Atlas Cloud
- **Security**: Bcrypt, JSON Web Tokens (JWT)
- **Testing**: Jest

## Architecture Overview
This application is split into two main components: 
1. **Client Side**: Built with React, MUI, and Redux, providing a dynamic and responsive user interface.
2. **Server Side**: Utilizes Node Apollo with GraphQL, offering a robust and efficient back-end structure.

## Installation Requirements
Before running the client or server, ensure you have Node.js and npm (Node Package Manager) installed. Additionally, since this project uses MongoDB Atlas Cloud, ensure you have an internet connection for database access.

## Quick Start Guide
To get started with **Global Voices Technical Assignment**:
1. **Clone the Repository**: `git clone [repository-url]`
2. **Client**: Navigate to the client directory and follow the instructions in the [Client README](https://github.com/valgaug/my-auth-app/blob/master/client/README.md).
3. **Server**: Navigate to the server directory and follow the instructions in the [Server README](https://github.com/valgaug/my-auth-app/blob/master/server/README.md).
